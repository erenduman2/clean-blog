## Usage

First, in the directory, you can run:

### `npm install`

Then, you can run:

### `npm start`

Project will start in http://localhost:5000/.

## Tools

* Node.js
* Express.js
* Nodemon
* Mongoose
 

## Features

* Share Post
* Edit & Delete Posts
